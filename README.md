# type-feel

what if we could see the shape of our feelings, the ebbs and flows of our moods more clearly?

playing with variable fonts. prototyping a visual mood journal. dynamically mapping moods to text using the [circumplex model](https://en.wikipedia.org/wiki/Emotion_classification#Circumplex_model).
